  --> 下面是所有的指令以及其对应的功能:
  --> env : 查看身边的环境参数
      change location : 改变你的所在地参数
      write diary : 写一下今天的日记
      read diary :读一篇之前写过的日记
      begin protection : 开启保护模式
      stop protection :关闭保护模式
      warning report : 查看入侵记录
      alarm : 查看目前设定的所有闹钟，并进行添加或者删除
      set alarm : 设定闹钟铃声
      exit it : 关闭远程设备
  --> 就这些 :)
